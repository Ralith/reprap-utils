reprap-utils by Ralith

Various lightweight UNIXy command-line utilities for working with RepRap machines on POSIX-compatible systems and Windows.

Dependencies

Runtime: none
Buildtime: CMake

Install

From source: cmake . && make && make install

From binary: No installation necessary.

Authors

Benjamin Saunders

Download

Precompiled Linux binaries are available and believed to be stable. Although the utils are intended to be portable, windows support is largely unmaintained; only some outdated and untested binaries are available for that platform.

You can download the source code of this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/Ralith/reprap-utils
